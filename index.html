Tables responsive

Easy customizable, framework agnostic, responsive table patterns written with SASS and easy to use with Twitter Bootstrap

Basic one - revert whole table and add scroll for table body

Pretty simple to use, just add .table-revert class to table. No need to use any additional containers, just switching display mode for <thead>, <tbody> and <tr>

A little bit javascript - hide few columns, and let user choose what he wants to see

To use it, class .hidden-col should be added for all unnecessary columns. And .toggle-col class with data attribute data-target="#toggle-cols" to set-up dropdown container, for a table
